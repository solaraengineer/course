üíª What is HTML?

HTML stands for HyperText Markup Language. It's not a programming language - it's a markup language that tells browsers how to structure and display content. Every website you visit is built with HTML at its core.

HTML uses a system of tags to structure content. Tags are keywords surrounded by angle brackets, and most come in pairs with an opening and closing tag. The content goes between these tags.

Basic HTML Structure

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>

Breaking Down the Structure

<!DOCTYPE html> - This declaration tells the browser that this is an HTML5 document. It must be the very first thing in your HTML file.

<html> - This is the root element that contains all other HTML elements. The lang attribute specifies the language of the document.

<head> - Contains metadata about the document like the title, character encoding, and links to stylesheets. This content isn't visible on the page.

<body> - Contains all the visible content of your web page - text, images, links, and more.

üí° Pro Tip

Always use semantic HTML tags like <header>, <nav>, <main>, <article>, and <footer> instead of generic <div> tags. This makes your code more accessible, improves SEO, and makes it easier for other developers to understand your structure!

üî§ Common HTML Tags

Headings

HTML provides six levels of headings, from <h1> to <h6>. Use <h1> for your main page title and use the others in hierarchical order.

<h1>Main Title</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>

Paragraphs and Text

The <p> tag defines a paragraph. Use <strong> for important text (bold) and <em> for emphasized text (italic).

<p>This is a paragraph with <strong>bold</strong> and <em>italic</em> text.</p>

Links

The <a> tag creates hyperlinks. The href attribute specifies where the link goes.

<a href="https://example.com">Visit Example</a>

Images

The <img> tag embeds images. It's a self-closing tag and requires the src attribute for the image source and alt for accessibility.

<img src="photo.jpg" alt="Description of the image">

‚ö†Ô∏è Important

Always include an alt attribute on images! This is crucial for accessibility - screen readers use this text to describe images to visually impaired users.

üìù Lists

HTML supports two types of lists: unordered (bullet points) and ordered (numbered).

Unordered List

<ul>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>

Ordered List

<ol>
  <li>Step one</li>
  <li>Step two</li>
  <li>Step three</li>
</ol>

üî• Practice Exercise

Try creating a simple HTML page about yourself. Include:

  • A main heading with your name
  • A paragraph introducing yourself
  • An unordered list of your hobbies
  • An ordered list of your top 3 favorite movies
  • A link to your favorite website

Don't worry about styling yet - we'll cover CSS in the next lesson. For now, focus on getting the structure right and using the appropriate HTML tags.

üí° Challenge Yourself

Once you've completed the basic exercise, try adding semantic HTML elements like <header>, <main>, and <footer> to better structure your page. This is what professional developers do!

üéì Key Takeaways

  • HTML is the foundation of all web pages
  • Tags come in pairs (opening and closing) with content in between
  • Always use semantic HTML for better accessibility and SEO
  • The basic structure includes DOCTYPE, html, head, and body tags
  • Common tags include headings, paragraphs, links, images, and lists
  • Always include alt text on images for accessibility
