üìñ Introduction

Welcome to DOM Manipulation where JavaScript gets REALLY powerful! The DOM (Document Object Model) is how JavaScript sees and interacts with your HTML Think of it as a bridge between your code and your webpage. With DOM manipulation you can change content, styles, and structure dynamically without reloading the page.

This is what makes modern websites feel alive content updates, interactive forms, animations, and everything that responds to user actions. By the end of this lesson, you'll be able to make your websites truly interactive!

üå≥ What is the DOM?

The DOM (Document Object Model) is a programming interface for web documents. It represents your HTML as a tree structure that JavaScript can access and modify. Every HTML element becomes a "node" that you can select, change, or manipulate.

When your browser loads a page it creates the DOM automatically. JavaScript can then access this DOM and make changes that instantly appear on the page no refresh needed! thats why DOM will be a powerfull function.

üí° Think of it this way

HTML is the blueprint, CSS is the paint, and JavaScript with DOM manipulation is like having a remote control that can rebuild and repaint the house while you're living in it!

üéØ Selecting Elements

Before you can change anything, you need to select the HTML element. There are several ways to do this:

getElementById (Most Specific)

// HTML: <div id="myDiv">Hello</div>

const element = document.getElementById('myDiv');
console.log(element); // Gets the div

querySelector (Most Flexible)

// Select by ID
const byId = document.querySelector('#myDiv');

// Select by class
const byClass = document.querySelector('.myClass');

// Select by tag
const byTag = document.querySelector('p');

// Complex selectors work too!
const complex = document.querySelector('div.container > p');

querySelectorAll (Get Multiple Elements)

// Get all paragraphs
const allP = document.querySelectorAll('p');

// Loop through them
allP.forEach(p => {
  console.log(p.textContent);
});

‚ö†Ô∏è Common Mistake

querySelector returns the FIRST!! matching element If you want all elements use querySelectorAll Also, if no element is found, you'll get null or None (varies on scenario) always check before using the element..

‚úèÔ∏è Changing Content

Once you've selected an element, you can change its content in several ways:

textContent (Plain Text Only)

const element = document.getElementById('myDiv');

// Get current text
console.log(element.textContent);

// Set new text
element.textContent = 'New text here';

innerHTML (Can Include HTML)

const element = document.getElementById('myDiv');

// Add HTML content
element.innerHTML = '<strong>Bold text!</strong>';

// Append to existing content
element.innerHTML += '<p>New paragraph</p>';

‚ö†Ô∏è Security Warning

Be careful with innerHTML when using user input - it can be vulnerable to XSS attacks. For user-generated content, use textContent instead or properly sanitize the input!

üé® Changing Styles

You can change CSS styles directly with JavaScript!

Inline Styles

const element = document.getElementById('myDiv');

// Change one style
element.style.color = 'red';
element.style.fontSize = '20px';
element.style.backgroundColor = '#333';

// Note: CSS property-name becomes camelCase propertyName

CSS Classes (Better Practice)

const element = document.getElementById('myDiv');

// Add a class
element.classList.add('active');

// Remove a class
element.classList.remove('hidden');

// Toggle a class (add if not there, remove if there)
element.classList.toggle('dark-mode');

// Check if class exists
if (element.classList.contains('active')) {
  console.log('Element is active!');
}

üí° Pro Tip

Using classes is better than inline styles because it keeps your styling in CSS where it belongs. Define your styles in CSS, then just add/remove classes with JavaScript!

‚ûï Creating & Removing Elements

You can create new HTML elements and add them to the page dynamically!

Creating Elements

// Create a new div
const newDiv = document.createElement('div');

// Add content
newDiv.textContent = 'I am new!';

// Add classes
newDiv.classList.add('my-class');

// Add to page
document.body.appendChild(newDiv);

Removing Elements

const element = document.getElementById('myDiv');

// Remove the element
element.remove();

// Or remove from parent
element.parentNode.removeChild(element);

üëÇ Event Listeners

Event listeners let you run code when something happens a click, hover, keypress, etc. This is how you make things interactive

Click Events

const button = document.getElementById('myButton');

button.addEventListener('click', () => {
  console.log('Button clicked!');
});

Common Events

const element = document.getElementById('myDiv');

// Click
element.addEventListener('click', () => {});

// Mouse hover
element.addEventListener('mouseover', () => {});
element.addEventListener('mouseout', () => {});

// Keyboard
document.addEventListener('keydown', (e) => {
  console.log('Key pressed:', e.key);
});

// Form input
input.addEventListener('input', (e) => {
  console.log('Value:', e.target.value);
});

Event Object

button.addEventListener('click', (event) => {
  console.log(event.target); // The element clicked
  console.log(event.type); // Type of event
  event.preventDefault(); // Prevent default action
});

üéÆ Real Interactive Examples

‚ú® Counter Button

<!-- HTML -->
<button id="counterBtn">Count: 0</button>

<!-- JavaScript -->
<script>
  const btn = document.getElementById('counterBtn');
  let count = 0;

  btn.addEventListener('click', () => {
    count++;
    btn.textContent = `Count: ${count}`;
  });
</script>

‚ú® Show/Hide Toggle

<!-- HTML -->
<button id="toggleBtn">Toggle</button>
<div id="content">This can be hidden!</div>

<!-- CSS -->
<style>
  .hidden { display: none; }
</style>

<!-- JavaScript -->
<script>
  const btn = document.getElementById('toggleBtn');
  const content = document.getElementById('content');

  btn.addEventListener('click', () => {
    content.classList.toggle('hidden');
  });
</script>

‚ú® Todo List

<!-- HTML -->
<input type="text" id="todoInput" placeholder="New todo">
<button id="addBtn">Add</button>
<ul id="todoList"></ul>

<!-- JavaScript -->
<script>
  const input = document.getElementById('todoInput');
  const addBtn = document.getElementById('addBtn');
  const list = document.getElementById('todoList');

  addBtn.addEventListener('click', () => {
    const text = input.value.trim();
    if (!text) return;

    const li = document.createElement('li');
    li.textContent = text;
    list.appendChild(li);

    input.value = ''; // Clear input
  });
</script>

üéì Key Takeaways

  • The DOM is JavaScript's way of accessing and changing HTML
  • querySelector and getElementById are used to select elements
  • Use textContent for text, innerHTML for HTML content
  • classList.add/remove is better than inline styles
  • addEventListener makes things interactive
  • Always check if an element exists before using it
  • Practice is key - build small projects to learn!
