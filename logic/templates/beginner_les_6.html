üíª What is JavaScript?

JavaScript (JS) is a programming language that runs in web browsers It allows you to create dynamic content, control multimedia, animate images, handle user interactions, validate forms, and much more. Every time you see something move, update, or respond on a webpage that's JavaScript at work!

JavaScript is one of the three core technologies of the web, alongside HTML and CSS. It's also incredibly versatile you can use it for web development, mobile apps, server side programming, and even game development.

üì¶ Variables

Variables are containers for storing data. Think of them as labeled boxes where you can put information and retrieve it later.

Declaring Variables

// Modern way (use these!)
let name = "Solara";
const age = 25;

// Old way (avoid)
var city = "New York";

let - Use for variables that can change
const - Use for variables that won't change (constant)
var - Old way has weird scoping issues, avoid it

üí° Pro Tip

Always use const by default. Only use let when you know the value will change. This makes your code safer and easier to understand. Never use var in modern JavaScript!

üé≤ Data Types

JavaScript has several types of data you can work with:

Strings (Text)

const message = "Hello World";
const name = 'Solara';

Numbers

Booleans (True/False)

const numbers = [1, 2, 3, 4, 5];

‚ûï Basic Operators

Arithmetic Operators

let x = 10;
let y = 5;

console.log(x + y); // 15 (addition)
console.log(x - y); // 5 (subtraction)
console.log(x * y); // 50 (multiplication)
console.log(x / y); // 2 (division)
console.log(x % y); // 0 (remainder)

Comparison Operators

console.log(5 === 5); // true (equal value and type)
console.log(5 !== 10); // true (not equal)
console.log(10 > 5); // true (greater than)
console.log(5 < 10); // true (less than)
console.log(5 >= 5); // true (greater or equal)
console.log(10 <= 20); // true (less or equal)

‚ö†Ô∏è Important

Always use === (strict equality) instead of == (loose equality). The === operator checks both value and type, making your code more predictable and preventing weird bugs!

üéÆ Making Things Interactive

Let's put it all together with a real interactive example!

‚ú® Interactive Button Example

<!-- HTML -->
<button id="myButton">Click Me!</button>
<p id="output"></p>

<!-- JavaScript -->
<script>
  const button = document.getElementById('myButton');
  const output = document.getElementById('output');
  let clickCount = 0;

  button.addEventListener('click', () => {
    clickCount++;
    output.textContent = `Clicked ${clickCount} times!`;
  });
</script>

üìù Console.log() - Your Best Friend

The console.log() function prints information to the browser's console. It's essential for debugging and understanding what your code is doing!

console.log("Hello World");
console.log(5 + 3);
console.log(person);

// View console: Right-click page ‚Üí Inspect ‚Üí Console tab

üí° Debugging Tip

Use console.log() everywhere when learning! Print variables, function results, and anything you're unsure about. The console is where you'll spend a lot of time as a developer, and that's totally normal!

üéì Key Takeaways

  • JavaScript makes websites interactive and dynamic
  • Use const for constants, let for variables that change
  • Main data types: strings, numbers, booleans, arrays, objects
  • Functions are reusable blocks of code
  • Always use === for comparisons, not ==
  • console.log() is your debugging best friend
  • Practice is key - write code every day!
