CSS Styling Basics

📖 Reading Time: 20 min
📊 Beginner
👨‍🏫 Solara

📖 Introduction

Now that you understand HTML structure, it's time to make your websites look amazing! CSS (Cascading Style Sheets) is what transforms plain HTML into beautiful, engaging web pages. If HTML is the skeleton, CSS is the skin, clothes, and makeup.

In this lesson, we'll dive into the fundamentals of CSS and learn how to style your web pages like a pro. Get ready to bring color, layout, and personality to your websites!

🎯 What You'll Learn

  • Understanding CSS syntax and how it works
  • Different ways to add CSS to your HTML
  • CSS selectors and how to target elements
  • Common CSS properties for styling text and layout
  • The box model and how spacing works
  • Colors, fonts, and basic design principles

💻 What is CSS?

CSS stands for Cascading Style Sheets. It's a language used to describe how HTML elements should be displayed on screen. CSS controls everything visual: colors, fonts, spacing, layout, animations, and more.

The "Cascading" part means that styles can inherit and override each other in a specific order - we'll get into that later!

CSS Syntax

CSS consists of rulesets. Each ruleset has a selector (which element to style) and declarations (what styles to apply).

selector {
  property: value;
  another-property: value;
}

✨ Real Example

h1 {
  color: purple;
  font-size: 32px;
  text-align: center;
}

🔗 Adding CSS to HTML

There are three ways to add CSS to your HTML pages:

1. Inline CSS

Applied directly to HTML elements using the style attribute. Quick but not recommended for larger projects.

<h1 style="color: blue; font-size: 24px;">Hello World</h1>

2. Internal CSS

Written inside a <style> tag in the HTML head section. Good for single-page styling.

<head>
  <style>
    h1 {
      color: blue;
      font-size: 24px;
    }
  </style>
</head>

3. External CSS (Recommended!)

Written in a separate .css file and linked to your HTML. This is the professional way - it keeps your code organized and allows you to reuse styles across multiple pages.

<head>
  <link rel="stylesheet" href="styles.css">
</head>

💡 Best Practice

Always use external CSS files for your projects! It makes your code cleaner, easier to maintain, and allows you to apply the same styles across multiple pages. Professional developers almost never use inline or internal CSS.

🎯 CSS Selectors

Selectors tell CSS which HTML elements to style. Here are the most common ones:

Element Selector

Targets all elements of a specific type.

p {
  color: white;
}

Class Selector

Targets elements with a specific class attribute. Use a dot (.) before the class name.

.button {
  background-color: purple;
  padding: 10px 20px;
}

<!-- HTML -->
<button class="button">Click Me</button>

ID Selector

Targets a single element with a specific ID. Use a hash (#) before the ID name.

#header {
  background-color: black;
}

<!-- HTML -->
<div id="header">My Header</div>

⚠️ Important Rule

IDs should be unique! Only one element on a page should have a specific ID. For styling multiple similar elements, always use classes instead. IDs are best reserved for JavaScript or linking to specific page sections.

🎨 Common CSS Properties

Colors

CSS supports multiple color formats:

/* Named colors */
color: red;
background-color: white;

/* Hex codes */
color: #7c3aed;

/* RGB */
color: rgb(124, 58, 237);

/* RGBA (with transparency) */
background-color: rgba(124, 58, 237, 0.5);

Typography

font-family: Arial, sans-serif;
font-size: 16px;
font-weight: bold;
text-align: center;
line-height: 1.6;
letter-spacing: 1px;

Spacing

margin: 20px; /* Space outside element */
padding: 10px; /* Space inside element */
margin-top: 15px; /* Specific side */
padding: 10px 20px; /* Top/bottom left/right */

📦 The Box Model

Every HTML element is essentially a box. The CSS box model describes how these boxes are sized and spaced. From inside out, each box consists of:

  • Content - The actual text or images
  • Padding - Space between content and border
  • Border - A line around the padding
  • Margin - Space outside the border

✨ Box Model Example

.box {
  width: 200px;
  padding: 20px;
  border: 2px solid purple;
  margin: 10px;
}

/* Total width = 200 + (20*2) + (2*2) + (10*2) = 264px */

💡 Pro Tip

Use "box-sizing: border-box;" on all elements to make width calculations easier. This includes padding and border in the width calculation, making layouts much more predictable!

🎨 Styling Borders and Backgrounds

/* Borders */
border: 2px solid black;
border-radius: 10px; /* Rounded corners */
border-top: 3px dashed red;

/* Backgrounds */
background-color: purple;
background-image: url('image.jpg');
background-size: cover;

🔥 Practice Exercise

Create a simple card component using everything you've learned:

  • Make a div with class "card"
  • Give it a background color, padding, and rounded corners
  • Add a heading and paragraph inside
  • Style the heading with a different color and larger font
  • Style the paragraph with good line-height for readability
  • Add a border and some margin around the card

✨ Challenge Solution Example

.card {
  background-color: white;
  padding: 30px;
  border-radius: 15px;
  border: 2px solid #7c3aed;
  margin: 20px;
  box-shadow: 0 4px 10px rgba(0,0,0,0.2);
}

.card h2 {
  color: #7c3aed;
  font-size: 24px;
  margin-bottom: 10px;
}

.card p {
  color: #333;
  line-height: 1.6;
}

🎓 Key Takeaways

  • CSS controls all visual styling on websites
  • Always use external CSS files for clean, maintainable code
  • Classes are for styling multiple elements, IDs for unique elements
  • The box model consists of content, padding, border, and margin
  • Color, typography, and spacing are fundamental to good design
  • Practice is key - experiment with different properties!
